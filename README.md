# Find Your Own Skills

## 中文

这是一个用于“找技能、选技能、再执行”的 Skill 仓库。

仓库里包含两个版本：
- `SKILL.md`：给 Codex 使用
- `.claude/skills/find-your-own-skills/SKILL.md`：给 Claude Code / Claude Desktop 使用

它们的作用都一样：只搜索已安装的本地 skills，展示匹配结果、功能简介和路径，然后让用户选择一个再继续。

## English

This repository provides a skill for "find, compare, then use" workflows.

It includes two versions:
- `SKILL.md` for Codex
- `.claude/skills/find-your-own-skills/SKILL.md` for Claude Code / Claude Desktop

Both versions do the same thing: search only installed local skills, show ranked matches with short function summaries and paths, then ask the user to choose one before continuing.
