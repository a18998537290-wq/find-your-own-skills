# Contributing

Thanks for helping improve Find Your Own Skills.

## Good Contributions

- Improve matching quality.
- Add examples for real skill libraries.
- Improve Chinese or English documentation.
- Add support notes for more agent skill formats.
- Report cases where the wrong skill is ranked first.

## Pull Request Checklist

- Keep instructions concise.
- Test changed scripts with `python3 -m py_compile`.
- Keep Codex and Claude behavior aligned when possible.
- Update `CHANGELOG.md` for user-visible changes.

## 中文

感谢你帮助改进 Find Your Own Skills。

适合贡献的内容：

- 改进匹配质量。
- 添加真实 skill 库的示例。
- 改进中文或英文文档。
- 增加更多 agent skill 格式的支持说明。
- 报告错误排序或错误匹配的案例。

提交 PR 前请确认：

- 说明保持简洁。
- 修改脚本后运行 `python3 -m py_compile`。
- 尽量保持 Codex 和 Claude 行为一致。
- 用户可见改动同步更新 `CHANGELOG.md`。
